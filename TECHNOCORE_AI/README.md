AI Knowledge Assistant
A powerful AI chatbot application designed to help students and workers improve their knowledge across various domains.

Features

Multi-category assistance (General, Academic, Finance, Travel, Sports)
Voice input and output capabilities
File attachment support
Deep Think mode for more thoughtful responses
Conversation history and library
Responsive design for both mobile and desktop
Dark mode support

Tech Stack

Frontend

HTML5
CSS3
JavaScript (ES6+)

Backend

Java 11
Spring Boot
PostgreSQL
OpenAI API

Getting Started

Prerequisites

Java 11 or higher
Maven 3.6 or higher
PostgreSQL 12 or higher
Node.js (for frontend development)
